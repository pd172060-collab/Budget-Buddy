# **App Name**: Rupee Tracker

## Core Features:

- Transaction Input: Allow users to input transaction details (amount, date, category, description) in Rupees.
- Transaction Output: Display transaction history with clear filtering and sorting options.
- Login Page: Secure user accounts with login and registration functionality.
- Dark/Light Mode: Implement a toggle for users to switch between dark and light mode for optimal viewing experience.
- Data Storage: Save the data to a database

## Style Guidelines:

- Color scheme is dark.
- Primary color: A vibrant blue (#29ABE2) to convey trust and financial stability, working well against a dark background.
- Background color: Dark gray (#282C34), nearly desaturated blue.
- Accent color: A contrasting yellow (#F0E68C), a hue analogous to blue, to highlight key interactive elements such as buttons.
- Body and headline font: 'Inter', a sans-serif font, for clear readability across the application.
- Use consistent and clean icons to represent transaction categories.
- Subtle transitions and animations to provide feedback on user interactions.
